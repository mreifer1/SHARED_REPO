## HOW TO SET UP DEV ENVIRONMENT

Code taken retrieved from: https://github.com/groq/groq-api-cookbook/tree/main/tutorials/01-quickstart/chat-history <br />

create a file named .env in this directory <br />
create an environment variable called GROQ_API_KEY in the file <br />
once you type in GROQ_API_KEY= in the .env file, copy paste your API key after the = sign <br />

```sh
GROQ_API_KEY = <API_Key_String>
```
